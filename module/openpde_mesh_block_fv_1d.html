Concrete class of mesh block for Finite Volume 1D methods.

This mesh block is based on a uniform, Cartesian cell-centered discretization of the domain. The nodes-cell numeration adopts the following convention:

``` 0 1 2 n n+1 +-----+-----+… +-----+-----+-----+… +-----+-----+… +-----+ | | | | | | | | + + + + |-ng |-ng+1| | 0 | 1 | 2 | | n + n+1 + + n+ng+ | | | | | | | | + + + + +-----+-----+… +-----+-----+-----+… +-----+-----+… +-----+

``` where

+n is the number of cells +ng is the number of ghost cells

The cell size is uniform and equals to h.
